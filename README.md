# 🌍 TripMe - Canoa Quebrada

Projeto desenvolvido como **desafio de certificação do curso DevStart (HTML)**.
A proposta foi criar uma **página web temática de viagem**, contendo um banner principal e **três seções informativas** sobre um destino turístico.

O destino escolhido para o projeto foi **Canoa Quebrada**, uma das praias mais famosas do litoral do Ceará.

## 📖 Sobre o Projeto

O **TripMe** é uma landing page simples e responsiva que apresenta informações sobre o destino turístico **Canoa Quebrada**.

A página contém:

* Um **banner principal (Hero Section)** com imagem de fundo
* Menu de navegação com âncoras
* Seções informativas sobre o destino
* Cards com dicas para visitantes
* Botões de navegação para voltar ao topo

O objetivo do projeto foi **praticar conceitos fundamentais de HTML e CSS**, como:

* Estruturação semântica do HTML
* Navegação com âncoras
* Layout com **Flexbox**
* Estilização com **CSS**
* Uso de **background image**
* Organização visual de conteúdo

---

## 🧠 Desafio do Curso

Neste desafio do **DevStart**, o objetivo era criar uma página web que tivesse:

* Um **banner com imagem**
* Pelo menos **três seções principais**:

  * TripMe / Minha Viagem
  * MeetUs / Nos Encontre
  * Advice / Conselhos
* Navegação entre seções
* Não era obrigatório usar **JavaScript**

## 🌴 Sobre o Destino

**Canoa Quebrada** é um dos destinos turísticos mais famosos do estado do Ceará, conhecido por:

* Falésias coloridas
* Passeios de buggy nas dunas
* A famosa rua **Broadway**
* Pôr do sol incrível no litoral

A praia fica localizada no município de **Aracati - Ceará**, a aproximadamente **160 km de Fortaleza**.
